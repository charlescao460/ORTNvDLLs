# ORTNvDLLs
Windows Nvidia Runtime DLLs that ONNX Runtime required for inference.

## Versions
For ONNX Runtime version **1.17**

CUDA Version: `12.2.0`

cuDNN Version: `windows-x86_64-8.9.2.26_cuda12`

TensorRT Version: `8.6.1.6.Windows10.x86_64.cuda-12.0`
